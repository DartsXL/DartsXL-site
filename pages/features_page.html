Comprehensive guide to DartsXL Counter features
› Features Overview
DartsXL Counter is a free, no-download dart counter web app for playing 501 darts with any starting score and any darts in hand.
› Scoring Methods
Score by:
• 
Placing darts on an interactive dartboard.
• 
Enter a visit score.
• 
View checkout routes instantly.
Supports:
• 
Multiple players.
• 
Save/Restore. Switch between matches.
• 
Statistics and match history.
• 
Instant checkout routes for any score.
• 
Interactive, user-friendly interface.
› Checkout Routes Analyzer
Our advanced Checkout Routes Analyzer provides a sorted set of checkout finishing routes for any score or any dartboard. Providing opportunity for long-term improvement of strategy, and perfect fluency for all scenarios in a match of 501 darts.
The Checkout Routes Analyzer allows the user to:
• 
Display checkout routes from any historic point in a real match, via navigation.
• 
Input their own scores to generate checkout routes from.
• 
Navigate a history of presented routes in the analyzer.
• 
Create simulated matches based on any previous state in a real match. Allowing the user to see exactly what the scenario would have been like, if they would have made different decisions at any point in the match.
• 
Navigate an advanced, interactive GUI supporting four dartboards.
› Multiple Dartboard Layouts
